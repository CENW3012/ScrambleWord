## How to Play

1. Unscramble the letters to form the correct word
2. Read the clue to help you guess
3. Type your answer and click Submit
4. Use Hint for help (costs 5 points)
5. Skip if you're stuck
6. Complete all words to win!

## Setup

1. Clone this repository
2. Add your media files to the assets folders
3. Open `index.html` in your web browser

## Customization

- Add more words to the `words` array in `script.js`
- Replace sound files in `assets/sounds/`
- Replace background image in `assets/images/`
